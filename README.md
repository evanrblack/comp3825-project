# UDP File Server for COMP3825

Requires Ruby

## To start the server

```ruby udp_server.rb <log level>```

The log level is optional


## To run the client

```ruby udp_client.rb <ip> <protocol> <file>```

Valid protocols include 'ab' (alternating-bit) and 'gbn' (go-back-n)
